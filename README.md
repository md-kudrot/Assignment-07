# KeenKeeper

KeenKeeper is a relationship tracker built with Next.js. It helps you keep track of friends, see who needs attention, and log interactions through a clean dashboard, timeline, and analytics view.

## Technologies Used

- Next.js 16
- React 19
- Tailwind CSS 4
- DaisyUI
- Recharts
- React Icons
- React Toastify

## Key Features

1. Friend dashboard with profile cards, contact status labels, and quick access to each friend’s detail page.
2. Individual friend pages with relationship stats, contact goals, and quick check-in actions for calls, texts, and video chats.
3. Timeline and analytics views that organize interactions by type and visualize activity with charts and filters.
